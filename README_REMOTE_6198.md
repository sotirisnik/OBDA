# Demo for the transformation of EHR data to I2B2 using MipMap

## Introduction

This project creates a Docker image translating EHR demo data to I2B2 using MipMap.

## Usage

Run: `run.sh`

## Build

Run: `build.sh`

## Push on Docker Hub

Run: `./docker_push.sh`
